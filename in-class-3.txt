1. The ________ property allows you to specify the color of text inside an element. -----color

2. ________ can be defined as “the art and technique of arranging type in order to make
language visible”.------------------------------------------------------------------typography

3. The ________ CSS property sets how the content of a replaced element, such as an <img>
or <video>, should be resized to fit its container.--------------------------------object-fit

4. The _______ CSS property sets how an element is positioned in a document. The top, right,
bottom, and left properties determine the final location of positioned elements.------------position

5. Beyond positioning elements we also have access to ________ which set rules and
standards for how a layout should flow and position elements in a more systematic way.-----layout systems

6. CSS Flexible Box Layout, commonly known as Flexbox, is a _______ dimensional CSS web
layout model. --------------------------------------------------------------------------one

7. CSS Grid is a _______ dimensional CSS web layout model.-----------------------------two

8. A ________ is a collection of CSS stylesheets that are prepped and ready to use.-------framework
